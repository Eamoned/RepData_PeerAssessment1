(4) Imputting Missing values

####Calculate & report the total number of missing values in the dataset.

sum(rowSums(is.na(activityData)))

## [1] 2304

####Devise a strategy for filling in all missing values in the dataset and create a new dataset that is equal to the original dataset but with the missing data filled in.

Strategy: Fill in missing values with the mean for that 5-minute interval and create a new dataset.

naData <- is.na(activityData$steps)
meanInterval <- tapply(activityData$steps, activityData$interval, mean, na.rm=TRUE)
newDataset <- activityData
newDataset$steps[naData] <- meanInterval[as.character(newDataset$interval[naData])]

Check for missing values

any(is.na(newDataset$steps))

## [1] FALSE

Histogram of total number of steps taken each day.

perDay <- tapply(newDataset$steps, newDataset$date, sum)

Discussion:

Although the the mean and median were approximately equal to begin with (data was normal and the distribution was approximately symetrical), the result of imputing missing data on the estimates does have an impact, in so far as the mean and median do converge after imputing. Also, now that the missing data is relaced by estimates (obviously estimated before imputing) the frequency of values closer to the mean has understandably increased.

So in summmary, weekends look a lot more active with more variation in the activities.
